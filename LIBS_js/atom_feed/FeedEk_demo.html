FeedEk jQuery RSS/ATOM Feed Plugin Examples

Basic Example

Code 
$('#divRss').FeedEk({
FeedUrl: 'https://www.jqueryscript.net/rss.xml',
MaxCount: 2
});

Description Character Limit Example

Code 
$('#divRss').FeedEk({
FeedUrl: 'https://www.jqueryscript.net/rss.xml',
MaxCount: 3,
ShowDesc: true,
ShowPubDate: false,
DescCharacterLimit: 100
});

No Description Example

Code 
$('#divRss').FeedEk({
FeedUrl: 'https://www.jqueryscript.net/rss.xml',
MaxCount: 4,
ShowDesc: false
});
